# 📂 案例 / Examples

真实的端到端使用案例，展示企业中枢处理完整需求单的全流程。

| # | 案例 | 需求单内容 | 涉及部门 | 复杂度 |
|---|------|---------|----------|--------|
| 1 | [竞品分析](competitive-analysis.md) | "分析 CrewAI vs AutoGen vs LangGraph" | 中书→门下→财务部+运维部+公关部 | ⭐⭐⭐ |

- 📋 **PMO规划**：任务拆解方案
- 🔍 **风控部审核**：审核意见（含打回重做/返工记录）
- ⚔️ **各部执行结果**：每个部门的产出摘要

> 💡 这些案例基于真实运行记录整理，任务 ID 和时间戳已脱敏。
